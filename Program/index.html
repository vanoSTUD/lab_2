Лабораторная работа №2

На тему: "HTML5 и веб-формы"

Цель работы

Практическое освоение возможностей языка HTML5 для описания страниц и веб-форм.

Задание

Создать семантическую разметку страницы с использованием возможностей, включённых в язык разметки HTML5.

Страница должна содержать
Заголовочная часть страницы:
  1. Логотип.
  2. Название страницы.
  3. Набор навигационных элементов (навигация по закладкам на странице).
Меню (навигация по закладкам на странице).
Основная часть.
  1. Несколько фрагментов текста с заголовками.
  2. Дополнительные блоки информации, относящиеся к основному материалу фрагментов.
  3. Форма для опроса, предполагающая ввод следующих видов данных:
    • текстовые (однострочные и многострочные);
    • числовые из некоторого диапазона;
    • дата;
    • телефон;
    • адрес электронной почты;
    • URL;
    • цвет.
Подвал страницы, содержащий не менее двух списков понятий с заголовками (например, «О сайте», «Правовая информация»).
